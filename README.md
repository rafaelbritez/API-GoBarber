# API para agendamento de servicos

 O intuito do projeto foi construir estrutura Back-End utilizando Node.js para agendamentos de serviços(GoBarber), aplicando o modelo de MVC, middleware de autenticação e banco de dados programavel por Javascript.


Foram usados algumas ferramentas como, Docker para container de imagens; EsLint e Prettier para padronização do projeto e correções automaticas; Sequelize para configuração, criação de tabelas e manipulação de banco de dados (PostgreSQL )utilizando sintaxe JavaScript;JWT para autenticação criptografia de dados; Redis banco de dados não relacional para requisições de cancelamento de agendamento enviando para o email;MongoDB como banco de dados não relacional e armazenamento de notficações; Insomnia para testar as requisições Front-End em Json.
